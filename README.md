# ED140023 Deep Learning for PDEs Project

This repository contains the implementation notebooks, result histories, datasets, and figures for the exam project.

## Datasets

The datasets used for the project are stored in the `Datasets` folder.

- `Datasets/ProblemA_dataset.h5`: dataset for Problem A.
- `Datasets/ProblemB_dataset.h5`: dataset for Problem B.
- `Datasets/ProblemC_dataset.h5`: dataset for Problem C.

## Problem A

- `Problem_A.ipynb`: working/development notebook for Problem A.
- `Problem_A_Results/Problem_A_Final.ipynb`: final reported run used in the report.
- `Problem_A_Results/Problem_A_Result_1.ipynb` to `Problem_A_Result_4.ipynb`: supplementary hyperparameter trials.
- `Problem_A_Results/Figures`: folder filled with all figures from the notebook relevant for Problem A.

The report links directly to `Problem_A_Results/Problem_A_Final.ipynb` as the original code for Problem A, unless stated otherwise.
